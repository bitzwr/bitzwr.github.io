HashMap

发表于 2018-06-09 | 分类 Java
| 总共 310 字 | 大约需要 1 分钟

Hash (哈希、散列)是计算机科学中一种对数据处理的方法，通过hash可以从任何数据中生成一个小的数字“指纹”，这个数字“指纹”就是哈希值。Hash 有一个基本特性，就是如果两个哈希值不一样，那么两个哈希值的原始数据肯定不一样。但是并不能保证哈希值一样，原始数据就一定一样。也就是说，不同的数据的哈希值有可能是一样的。这种现象就称为冲突。

Hash 在计算机领域被广泛应用，本文主要介绍哈希表。

初识哈希表

还记得大学时学习数据结构课程，其中哈希表的定义：

根据设定的哈希函数和处理冲突方法将一组关键字映像到一个有限的连续的地址集上并以关键字在地址集中的“像”作为记录在表中的存储位置，这种表便称为哈希表，这一映象过程称为散列，所得存储位置称哈希地址。
